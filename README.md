# Rawdhat HR Management System

A comprehensive Human Resources Management System (HRMS) built with Laravel, designed to streamline HR operations for educational institutions.

## Features

- **Employee Management**: Track employee details, contracts, and documents.
- **Leave Management**: Request, approve, and track employee leave.
- **Attendance Tracking**: Monitor daily attendance and working hours.
- **Payroll Management**: Generate payslips and manage salary information.
- **Recruitment**: Manage job postings and candidate applications.
- **Performance Management**: Set goals and track employee performance.
- **User Management**: Role-based access control for different departments.

 

 

Access the application at: [http://localhost:3000](http://localhost:5000)
